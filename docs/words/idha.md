# *idha*

## Dictionaries

### PTS Pāli-English Dictionary

- Indeclinable
- Sanskrit = *iha*
- Meaning = 
    - here
    - in this place
    - in this connection
    - now
    - (Especially in this world or present existence)
- On different meanings of *idha* see DhsA 348.

### Margaret Cone's A Dictionary of Pāli

- Indeclinable
- Sanskrit = *iha*
- Meaning =
    - here, in this place
    - hither
    - in this world
    - in this existence
    - in this case, in this connection (introducing an example or hypothesis)
    - in this teaching, in this system

### Monier Monier-Williams's A Sanskrit Dictionary

- Sanskrit *= ihá*
- indeclinable, from pronominal base 3.i
- Meaning:
    - in this place, here; 
    - to this place; 
    - in this world; 
    - in this book or system; 
    - in this case;
    - now, at this time
